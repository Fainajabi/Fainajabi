GOJEK Parking lot Problem

### Project Requirements

* JDK 1.8.

* Maven 3.1.0

* Junit 4.12

The project can be run as follow :
./parking_lot.sh  <input_filepath>  
   The inputs commands are expected and taken from the file specified
   
   Or else
